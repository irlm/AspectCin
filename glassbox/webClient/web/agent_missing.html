Glassbox Agent jar either not installed or not on the system classpath


Troubleshooting your installation....
  • Did you setup Glassbox on your Application Server? (Tomcat, WebSphere, etc)
  • Did you restart your application server?
  • Check the your Application Server's home directory to verify the agent.jar file is present in the glassbox directory
  • Add the agent.jar file as the first entry in your application server's system classpath. Consult your server documentation for more info
  • Check on the Glassbox Forums
  • Contact Glassbox Support
